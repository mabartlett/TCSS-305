Week 7

GUI Components

All GUI components need a JFrame object onto which the components will be displayed. Here is an example of creating a JFrame:

final JFrame window = new JFrame("Window Title");
window.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
window.setSize(640, 480);
window.setVisible(true);
window.setLayout(null);

Panels are added to group components. You can add components to either the frame or to the panel with the add(Component) method. We set bounds of a component by using the setBounds(x, y, width, height) method.

A label is a String that’s displayed somewhere for information or description. You cannot edit it. Similar to a label is a JTextField, which allows for editing. A JTextField has multiple lines. For JCheckBoxes, each can be checked or unchecked. Among a group of JRadioButtons, only one should be made selectable. This feature is not native to JRadioButton. You have to code it yourself!

Set layout to null for all containers unless you are using a layout. Setting it to null means you have to use absolute positioning for all components.

There are several types of JOptionPanes . The first is showMessageDialog, which simply dislays a message. The second is showConfirmDialog, which displays a message and the buttons “yes”, “no”, and “cancel.” The third is showInputDialog, which displays a message as well as a text field for the user to input text. The first argument for JOptionPane should be the JFrame object or else an error will be thrown.

Layout Managers

Absolute positioning is problematic and time-consuming, so we use the setLayout(LayoutManager) method to set a layout manager. Here are the LayoutManagers:

  • BorderLayout, which divides the container into 5 regions: NORTH, SOUTH, WEST, EAST, CENTER (all of which are constants of BorderLayout).
  • GridLayout
  • FlowLayout
  • BoxLayout
  • GridBagLayout

validate() checks and fixes the layout when you change the components of an already on-screen container. repaint() inherits from Component. It forces a redraw of all components in the container. The Pack() method uses the preferred sizes.

Each component has a preferred size.

Other Important Notes

Surround the Scanner with try/catch block with finally. Catch the exception inputMismatchException.

Declare a public static final field for Scanner in Registration and pass that to each method. Do not create multiple instances of Scanner because it will be incredibly messy and managing them all will be incredibly difficult.
